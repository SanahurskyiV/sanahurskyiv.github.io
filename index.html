Responsive layout in 2018.
CSS Grid

Велика кількість систем
побудови розмітки.

Flexbox, Bootstrap

  • одновимірність, розкладка тільки в одному напрямку
  • необхідність створювати нові контейнери-wrapperи
  • верстка підлаштовується під контент, а не навпаки
  • загромадженість в html розмітці

Що там у браузерів?

Крок 1

Перетворюємо нашу обгортку в сітку


        .wrapper {
  display: grid;
}

Крок 2

Малюємо віртуальну
сітку



          .wrapper {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 50px 50px;
}

Одиниці вимірювання

  • auto
  • px, rem, em, vh, vw і т. д.
  • %
  • fr
  • max-content, min-content, minmax(min, max)

    • 1vh = 1% від висоти вікна. При зменшенні висоти вікна, зменшується ширина, висота, шрифт елемента.
