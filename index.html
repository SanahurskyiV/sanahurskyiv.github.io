побудови розмітки.

Flexbox, Bootstrap

  • одновимірність, розкладка тільки в одному напрямку

  • верстка підлаштовується під контент, а не навпаки
  • загромадженість в html розмітці

Одиниці вимірювання

  • auto
  • px, rem, em, vh, vw і т. д.
  • %
  • fr
  • max-content, min-content, minmax(min, max)

    • 1vh = 1% від висоти вікна. При зменшенні висоти вікна, зменшується ширина, висота, шрифт елемента.

  .container {
    display: grid;
    grid-gap: 5px;    
    grid-template-columns: repeat(12, 1fr);
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        ". h h h h h h h h h h ."
        "c c c c c c c c c c m m"
        ". f f f f f f f f f f .";
}
 

  .header {
    grid-area: h;
}
.menu {
    grid-area: m;
}
.content {
    grid-area: c;
}
.footer {
   grid-area: f;
}

Додаємо адаптивність

  @media screen and (max-width: 640px) {
    .container {
        grid-template-areas:
            "m m m m m m h h h h h h"
            "c c c c c c c c c c c c"
            "f f f f f f f f f f f f";
    }
}
